# dubbing-workbench Spec Delta

## ADDED Requirements

### Requirement: 检查员模式

以流水线上下文（任务与文件标识）打开工作台时 SHALL 进入检查员模式：顶部上下文条显示任务名与待检文件序号（第 x / 共 N 个），提供「上一个/下一个」（切换同任务其它待校对文件的会话，清单每次动作前实时获取）、「放行并继续」（放行当前文件，有下一个则自动切换，否则返回任务详情）与「全部放行」（二次确认）；检查员模式下 MUST 隐藏「导出」入口（产出由任务的合成阶段负责），行级试听/重生成/换音色/接受变速等既有操作照常可用。

#### Scenario: 放行并继续

- **WHEN** 检查员模式下用户对当前文件点击「放行并继续」且还有 2 个待检文件
- **THEN** 当前文件放行续跑，工作台自动切换到下一个待检文件的会话

#### Scenario: 最后一个放行返回任务

- **WHEN** 检查员模式下放行最后一个待检文件
- **THEN** 工作台返回该任务详情页，任务列表可见文件已进入合成阶段

#### Scenario: 检查员模式无导出

- **WHEN** 以检查员模式打开工作台
- **THEN** 导出按钮不出现，成片由任务的合成阶段自动产出
