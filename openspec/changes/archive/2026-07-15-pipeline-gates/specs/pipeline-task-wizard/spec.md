# pipeline-task-wizard Spec Delta

## ADDED Requirements

### Requirement: 人工把关开关

向导在勾选配音或成品视频任一目标时 SHALL 显示「人工把关」区：`字幕校对` 开关（默认**开启**——下游存在成本型阶段时停一下有真实价值）与 `配音确认` 开关（仅勾选配音时显示，默认关闭）；文案白话并附一句代价说明（如「到点会通知你，校对后自动继续」）。开关取值随任务配置快照生效；均关闭时任务全自动。

#### Scenario: 默认值

- **WHEN** 用户勾选「配音 + 成品视频」
- **THEN** 人工把关区出现，字幕校对默认开启、配音确认默认关闭

#### Scenario: 纯字幕任务不显示把关

- **WHEN** 用户仅勾选「翻译字幕」（无配音/成品视频）
- **THEN** 不显示人工把关区，任务行为与 P2 一致

#### Scenario: 全自动可选

- **WHEN** 用户把两个开关都关闭并开始任务
- **THEN** 全流程自动执行到成片，无停靠
