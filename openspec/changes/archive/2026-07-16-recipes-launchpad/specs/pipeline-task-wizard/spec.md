# pipeline-task-wizard Spec Delta

## ADDED Requirements

### Requirement: 存为配方

向导 SHALL 提供「存为配方」入口：命名后把当前目标勾选、输入类型与全部阶段配置（字幕段/配音/合成/人工把关）保存为用户配方；保存成功即时反馈，配方出现在启动台。重名 SHALL 允许（配方以 id 区分）。

#### Scenario: 保存当前配置

- **WHEN** 用户在向导配好「翻译+配音+成品视频、配音确认开启」并存为「抖音出海」
- **THEN** 启动台出现「抖音出海」配方卡，下次应用时全部配置按保存时预填

### Requirement: 按配方预填

向导以 `?recipe=<id>` 打开时 SHALL 按配方预填：目标勾选、字幕段配置（浅合并到当前默认值之上，缺字段回落默认）、配音/合成配置与人工把关开关；配方引用的引擎/服务商失效时按既有就绪校验回落与内联引导。拖放交接的文件 SHALL 与配方预填共存。

#### Scenario: 配方预填与文件带入共存

- **WHEN** 用户往用户配方卡拖入视频，进入向导
- **THEN** 文件已带入、目标与各阶段配置按配方预填，可直接开始
